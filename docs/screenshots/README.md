## Como Capturar os Screenshots

### Recomendacoes

1. **Resolucao:** 1920x1080 ou superior
2. **Formato:** PNG (melhor qualidade)
3. **Modo:** Tema claro (melhor legibilidade no PDF)
4. **Dados:** Use dados de exemplo realistas

### Ferramentas Sugeridas

- **Windows:** `Win + Shift + S` (Recorte e Anotacao)
- **Mac:** `Cmd + Shift + 4`
- **Chrome:** Extensao "Full Page Screenshot"

### Tamanhos Recomendados

- **Telas completas:** 1200px largura
- **Modais:** 800px largura
- **Detalhes:** 600px largura

## Apos Adicionar as Imagens

1. Verifique se todas as imagens estao nomeadas corretamente
2. Otimize as imagens para web (compressao)
3. Converta o manual para PDF usando:
   - Pandoc: `pandoc MANUAL_USABILIDADE.md -o manual.pdf`
   - Notion: Importe o MD e exporte como PDF
   - Google Docs: Importe e exporte
